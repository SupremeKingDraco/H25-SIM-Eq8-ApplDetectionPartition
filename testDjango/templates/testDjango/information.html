Explication

Entrez le chemin absolu de votre vidéo et sélectionnez le mode de transcription souhaité.

Pour la transcription image : après avoir renseigné les paramètres, la vidéo s’affichera. Sélectionnez manuellement les notes à transcrire. Pour la sélection manuelle, faites en sorte de sélectionner, depuis la première note que vous avez mentionnée, uniquement les notes jouées. Cela veut dire que si la partition de musique contient un FA#, on ne sélectionne pas le FA naturel, mais seulement la touche noire. Puis appuyez sur Q pour valider.

Pour les autres méthodes (audio ou transposition), complétez les champs requis et cliquez sur Soumettre pour générer la partition. Pour la partie path pour le mp3, copier le debut du path absolute jusqu'a download et apres donnez un nom suivie de .mp3 tel que musique.mp3
